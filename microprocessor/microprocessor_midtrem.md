#### 4. ATmega2560 은 CISC 구조를 갖는다.

**답 : X, ATmega2560은 RISC 구조를 갖는다.**

- **RISC (Reduced Instruction Set Computer) 구조**
  - 명령어를 줄여 보다 빠른 처리 속도
  - CPU 구조 단순화
  - ex) Super Computer, ATmega
- **CISC (Complex Instruction Set Computer) 구조**
  - 명령어를 모두 내장 (CPU 구조 어렵다)
  - 뛰어난 호환성
  - Ex) 개인용 PC



#### 5. ATmega2560 은 데이터 영역과 프로그램 영역이 물리적으로 구분되어 있다.

**답 : O**

- **폰 노이만 구조**
  - 데이터 영역과 프로그램 영역의 물리적인 구분 X
  - ex) PC
- **하버드 구조**
  - 데이터 영역과 프로그램 영역의 물리적인 구분 O
  - ex) Arduino

#### 8. 마이크로 프로세서에서 명령어를 해석하고 실행하는데 필요한 컴퓨터 내부 장치 사이의 흐름을 제어하는 것은 연산장치이다.

 **답 : X, 제어 장치이다.**

- **마이크로 컴퓨터 구성**
  - **제어 장치 :** 명령어 해석과 실행
  - **연산 장치** : 연산 담당
  - **레지스터** : 데이터를 임시 저장, 고속 접근



#### 9. 일반적인 마이크로 컨트롤러는 대부분의 제어 기능이 하나의 IC 에 내장되어 있으므로 시스템을 변경하고자 할 경우에는 약간의 기능 변경과 추가를 통해 달성할 수 있어 타 시스템과의 이식성이 뛰어나다.

**답 : O**



#### 10. ATmega 2560 은 16 비트 마이크로 컨트롤러 이다.

**답 : X, 8 비트 마이크로 컨트롤러 이다.**

- **ATmega2560 은 저전력 CMOS 8 비트 마이크로 컨트롤러**



#### 11. ATmega2560 은 폰 노이만의 메모리 구조로 되어 있어 동시에 프로그램 메모리와 데이터 메모리를 액세스할 수 있다.

**답 : X, 폰 노이만 구조가 아니라 하버드 구조 이다.**

- 연산과 데이터 저장을 한번에 할 수 있는 **하버드 구조** 로 되어 있어 동시에 프로그램 메모리와 데이터 메모리를 동시에 액세스할 수 있다.

#### 14. ATmega2560 은 데이터 백업용의 메모리로서 4KB 의 EEPROM 을 내장하고 있다.

**답 : O**

- **4KB의 EEPROM** 을 내장하고 있어 데이터 백업이 가능함.

**답 : O**

#### 16. ATmega2560 에는 연산 결과를 AC(누산기) 레지스터에 저장한다.

**답 : X, AC가 없다.**

- 연산의 대상이 누산기(AC)가 아닌 이를 **32개의 범용 레지스터들을** 사용하여 연산을 할 수 있어서 **연산의 속도가 빠르다.**



#### 17. ATmega2560 에 대한 프로그램 메모리는 각 번지가 16 비트 구조로 되어 있어서 126K x 16 = 256KB 의 용량을 가진다.

**답 : O** 

- **용량 계산**

  ```
  126K x 16
  = 126K x 2^4
  = 126K x 2 x 2^3
  = 256K x Bytes
  = 256KB
  ```



#### 18. CISC 방식의 마이크로프로세서 장점은 뛰어난 호환성에 있다.

**답 : X, RISC 방식이다.**

- **CISC(Complex Instruction Set Computer)**
  - 명령어를 모두 내장 (CPU 구조 어렵다.)
  - 뛰어난 호환성
  - Ex) 개인용 PC
- **RISC(Reduced Instruction Set Computer)**
  - 명령어를 줄여 보다 빠른 처리 속도
  - CPU 구조 단순화
  - Ex) Super Computer, ATmega
