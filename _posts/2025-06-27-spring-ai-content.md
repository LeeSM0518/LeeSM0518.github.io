## 1. Ollama 환경 설정
---

brew를 사용해 `ollama` 를 설치한다.

```sh
brew install ollama
```

<br/>

ollama를 실행한다.

```sh
ollama start
```

<br/>

ollama로 deepseek-r1 모델을 다운받고 실행한다.

```sh
ollama run deepseek-r1
```

<br/>
